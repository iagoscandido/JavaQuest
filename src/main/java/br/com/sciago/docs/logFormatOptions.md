## Opções de formato para saída de log:

- ### Opção A - Detalhado (3 linhas):

```
Bob attacks Skeleton with Longsword
Attack: 15 (d20: 12 + modifier: +3) vs AC 13
Hit! Damage: 8 (d8: 5 + STR: +3)
```

- ### Opção B - Compacto (1-2 linhas):

```
Bob attacks Skeleton [Longsword] → Hit! (15 vs AC 13)
Damage: 8 (5+3)
```

- ### Opção C - Narrativo:

```
Bob swings their Longsword at Skeleton and hits! [Attack: 15 vs AC 13, Damage: 8]
```

- ### Opção D - Estilo D&D clássico:

```
Bob [Attack Roll: 15] → Skeleton [AC: 13] HIT!
Longsword damage: 8 (d8: 5 + modifier: 3)
```
